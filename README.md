# T-Coop

The enterprise platform for managing co-operative savings, contributions, loans, and dividends — rebuilt as a modern, premium SaaS frontend.

## Tech Stack

Next.js 15 (App Router) · React 19 · TypeScript · Tailwind CSS v4 · shadcn/ui · TanStack Query · React Hook Form + Zod · Zustand · Framer Motion · next-themes

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) — it redirects to `/login`.

## Scripts

- `pnpm dev` — start the dev server
- `pnpm build` — production build
- `pnpm lint` — run ESLint
- `pnpm format` — run Prettier

## Project Documentation

Per-page design decisions, accessibility notes, and architecture are documented under [`documentation/`](./documentation) as each screen ships.
